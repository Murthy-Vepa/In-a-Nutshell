    What are Data Structures?

    Data Structures are specialized formats for organizing, processing, and storing data efficiently. They enable efficient access and modification of data, which is crucial for designing efficient algorithms and software systems. The choice of the right data structure directly impacts the performance, scalability, and maintainability of your application.

    Why Data Structures Matter

    Understanding data structures is fundamental to software development for several reasons:

    ‚ö° Performance

    The right data structure can dramatically improve algorithm efficiency, reducing time complexity from O(n¬≤) to O(log n) or even O(1).

    üíæ Memory Optimization

    Efficient data structures minimize memory usage and fragmentation, critical for resource-constrained environments.

    üéØ Problem Solving

    Many complex problems become simple when the appropriate data structure is applied (e.g., graphs for networks).

    üìà Scalability

    Proper data structures ensure your application can handle growing data volumes efficiently.

    Choosing the Right Data Structure

    Consider these factors when selecting a data structure:

    • Operation Frequency: Which operations (insert, delete, search, update) are most common?
    • Data Size: How much data needs to be stored? Will it grow over time?
    • Access Pattern: Is access sequential, random, or based on priority?
    • Ordering Requirements: Do elements need to be sorted or maintain insertion order?
    • Memory Constraints: Are you optimizing for space or time?
    • Thread Safety: Will the structure be accessed by multiple threads?
    üí° Key Takeaway: There is no "best" data structure‚Äîonly the most appropriate one for your specific use case. Understanding the trade-offs between different structures is essential for building efficient, scalable applications.

    What is Complexity Analysis?

    Complexity Analysis is a systematic way of measuring how the runtime (time complexity) and memory usage (space complexity) of an algorithm grow as the input size increases. It helps us compare algorithms and choose the most efficient one for a given problem.

    Big O Notation

    Big O notation describes the upper bound of an algorithm's growth rate, focusing on the worst-case scenario. It ignores constants and lower-order terms to emphasize how the algorithm scales.

    üìê Formula: If an algorithm takes 3n¬≤ + 5n + 2 operations, its Big O notation is O(n¬≤) because the n¬≤ term dominates as n grows large.

    Detailed Complexity Examples

    O(1) - Constant Time

    Operations that take the same time regardless of input size.

    // Array element access
int value = array[5];  // O(1)

// Hash table lookup
value = hashMap.get("key");  // O(1) average case

// Stack/Queue operations
stack.push(item);      // O(1)
queue.dequeue();       // O(1)

    O(log n) - Logarithmic Time

    Algorithm halves the problem size in each step (divide and conquer).

    // Binary search
int binarySearch(int[] arr, int target) {
    int left = 0, right = arr.length - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (arr[mid] == target) return mid;
        if (arr[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}

    O(n) - Linear Time

    Time grows linearly with input size.

    // Array traversal
int sum = 0;
for (int i = 0; i < array.length; i++) {
    sum += array[i];  // O(n)
}

// Linear search
int linearSearch(int[] arr, int target) {
    for (int i = 0; i < arr.length; i++) {
        if (arr[i] == target) return i;
    }
    return -1;
}

    O(n log n) - Linearithmic Time

    Common in efficient sorting algorithms.

    // Merge sort
void mergeSort(int[] arr, int left, int right) {
    if (left < right) {
        int mid = (left + right) / 2;
        mergeSort(arr, left, mid);      // O(log n) depth
        mergeSort(arr, mid + 1, right);
        merge(arr, left, mid, right);   // O(n) at each level
    }
}

    O(n¬≤) - Quadratic Time

    Nested loops iterating over the same data.

    // Bubble sort
void bubbleSort(int[] arr) {
    for (int i = 0; i < arr.length; i++) {      // O(n)
        for (int j = 0; j < arr.length - 1; j++) { // O(n)
            if (arr[j] > arr[j + 1]) {
                swap(arr, j, j + 1);
            }
        }
    }
}

    Space Complexity

    Space complexity measures the total memory used by an algorithm, including:

    • Input space: Memory for the input data
    • Auxiliary space: Extra memory used by the algorithm (what we typically measure)
    • Output space: Memory for the output/result
    Space Complexity Description Example
    O(1) Constant space In-place sorting algorithms, iterative algorithms using few variables
    O(log n) Logarithmic space Recursive binary search (call stack depth)
    O(n) Linear space Creating a copy of array, hash table with n elements
    O(n¬≤) Quadratic space 2D matrix, adjacency matrix for graphs

    Best, Average, and Worst Case

    üéØ Best Case (Œ©)

    Omega notation. The minimum time/space an algorithm requires. Often not practical to rely on.

    üìä Average Case (Œò)

    Theta notation. Expected performance for typical inputs. Most realistic for practical analysis.

    ‚ö†Ô∏è Worst Case (O)

    Big O notation. Maximum time/space required. Critical for systems requiring guarantees.

    Array

    fixed contiguous indexed

    What is an Array?

    An array is a fixed-size, contiguous block of memory that stores elements of the same type. Elements are accessed using an index, with O(1) time complexity. Arrays are the simplest and most efficient data structure for random access.

    Key Characteristics

    • Fixed Size: Size must be declared at creation and cannot change
    • Contiguous Memory: Elements stored in adjacent memory locations
    • Index-Based Access: Direct access to any element using index
    • Type Homogeneous: All elements must be of the same type
    • Zero-Based Indexing: First element at index 0 (in most languages)

    Array Operations

    Search O(n) Linear scan required for unsorted array
    Insert (end) O(1)* If space available

    Delete O(n) Requires shifting elements

    ArrayList / Dynamic Array

    dynamic resizable growable

    What is an ArrayList?

    An ArrayList (or Dynamic Array) is a resizable array that automatically grows when capacity is exceeded. It maintains the O(1) access time of arrays while providing dynamic sizing. When the array is full, it allocates a larger array (typically 1.5x or 2x) and copies elements over.

    Key Characteristics

    • Dynamic Size: Automatically resizes as elements are added
    • Capacity vs Size: Maintains capacity (allocated) and size (used)
    • Amortized O(1) Insertion: At end, averaged over multiple operations
    • Contiguous Memory: Elements still stored contiguously
    • Index-Based Access: O(1) random access like arrays

    What is a Linked List?

    A Linked List is a linear data structure where elements (nodes) are stored in non-contiguous memory locations. Each node contains data and a reference (pointer) to the next node. Unlike arrays, linked lists don't require contiguous memory and can grow/shrink efficiently at any position.

    Types of Linked Lists

    ‚û°Ô∏è Singly Linked List

    Each node points to the next node. Traversal is one-directional (forward only).

    ‚¨ÖÔ∏è‚û°Ô∏è Doubly Linked List

    Each node has pointers to both next and previous nodes. Bi-directional traversal.

    üîÑ Circular Linked List

    Last node points back to the first node, forming a circle. Used in round-robin scheduling.

class Node:
    def __init__(self, data):

# Singly Linked List
class SinglyLinkedList:
    def __init__(self):
        self.head = None
    
    def add_first(self, data):
        """Add at beginning: O(1)"""
        new_node = Node(data)
        new_node.next = self.head
        self.head = new_node
    
    def add_last(self, data):
        """Add at end: O(n)"""
        new_node = Node(data)
        if not self.head:
            self.head = new_node
            return
        
        current = self.head
        while current.next:
            current = current.next
        current.next = new_node
    
    def remove_first(self):
        """Remove first: O(1)"""
        if not self.head:
            return None
        data = self.head.data
        self.head = self.head.next
        return data
    
    def search(self, data):
        """Search: O(n)"""
        current = self.head
        while current:
            if current.data == data:
                return True
            current = current.next
        return False
    
    def display(self):
        """Display all elements"""
        current = self.head
        while current:
            print(current.data, end=" -> ")
            current = current.next
        print("None")
    
    def __iter__(self):
        """Make iterable"""
        current = self.head
        while current:
            yield current.data
            current = current.next

        new_node = DNode(data)

            self.head.prev = new_node
            self.head = new_node

        new_node = DNode(data)
        if not self.tail:
            self.head = self.tail = new_node
        else:
            new_node.prev = self.tail
            self.tail.next = new_node
            self.tail = new_node

{

    foreach (char c in s)

    return new string(reversed);

class CircularQueue:
    def __init__(self, capacity):
        self.capacity = capacity
        self.items = [None] * capacity
        self.front = 0
        self.rear = -1
        self.count = 0
    
    def enqueue(self, item):
        if self.count == self.capacity:
            raise Exception("Queue is full")
        
        self.rear = (self.rear + 1) % self.capacity
        self.items[self.rear] = item
        self.count += 1

        if self.is_empty():

        return item

        if self.is_empty():
            raise Exception("Queue is empty")
        return self.items[self.front]
    
    def is_empty(self):
        return self.count == 0

        return self.count

// Custom Deque using Array (Circular Buffer)
public class ArrayDeque
{
    private int[] items;
    private int front;
    private int rear;
    private int count;
    private int capacity;
    
    public ArrayDeque(int capacity = 10)
    {
        this.capacity = capacity;
        items = new int[capacity];
        front = 0;
        rear = capacity - 1;
        count = 0;
    }
    
    public void AddFirst(int item)
    {
        if (count == capacity)
            throw new InvalidOperationException("Deque is full");
        
        front = (front - 1 + capacity) % capacity;
        items[front] = item;
        count++;
    }
    
    public void AddLast(int item)
    {
        if (count == capacity)
            throw new InvalidOperationException("Deque is full");
        
        rear = (rear + 1) % capacity;
        items[rear] = item;
        count++;
    }
    
    public int RemoveFirst()
    {
        if (count == 0)
            throw new InvalidOperationException("Deque is empty");
        
        int item = items[front];
        front = (front + 1) % capacity;
        count--;
        return item;
    }
    
    public int RemoveLast()
    {
        if (count == 0)
            throw new InvalidOperationException("Deque is empty");
        
        int item = items[rear];
        rear = (rear - 1 + capacity) % capacity;
        count--;
        return item;
    }
}

# Example: Sliding Window Maximum
def sliding_window_max(nums, k):
    """Find max in each window of size k"""
    result = []
    dq = deque()  # Store indices
    
    for i in range(len(nums)):
        # Remove elements outside window
        while dq and dq[0] <= i - k:
            dq.popleft()
        
        # Remove smaller elements
        while dq and nums[dq[-1]] < nums[i]:
            dq.pop()
        
        dq.append(i)
        
        if i >= k - 1:
            result.append(nums[dq[0]])
    
    return result

    What is a Tree?

    A Tree is a hierarchical data structure consisting of nodes connected by edges, with one node designated as the root. Unlike linear data structures (arrays, linked lists), trees organize data in a parent-child relationship, making them ideal for representing hierarchical data like file systems, organizational charts, and HTML DOM.

    Types of Trees

    üå≤ Binary Tree

    Each node has at most 2 children (left and right). Most common tree structure.

    üîç Binary Search Tree (BST)

    Binary tree with ordering: left child < parent < right child. Enables fast search O(log n).

    ‚öñÔ∏è AVL Tree

    Self-balancing BST maintaining height difference ‚â§ 1 between subtrees.

    üìä Heap

    Complete binary tree where parent is greater (max-heap) or smaller (min-heap) than children.

    Tree Traversal Methods

    Tree traversal is the process of visiting all nodes in a specific order:

    Traversal Order Use Case
    Inorder (LNR) Left ‚Üí Node ‚Üí Right BST: produces sorted output
    Preorder (NLR) Node ‚Üí Left ‚Üí Right Copy tree, prefix expression
    Postorder (LRN) Left ‚Üí Right ‚Üí Node Delete tree, postfix expression
    Level Order (BFS) Level by level, left to right Shortest path, serialization

    What is a Binary Tree?

    A Binary Tree is a tree data structure where each node has at most two children, referred to as the left child and right child. It's the foundation for more specialized trees like BST, AVL, and heaps.

    Types of Binary Trees

    • Full Binary Tree: Every node has 0 or 2 children (no node has only 1 child)
    • Complete Binary Tree: All levels filled except possibly the last, which fills from left
    • Perfect Binary Tree: All internal nodes have 2 children, all leaves at same level
    • Balanced Binary Tree: Height difference between left and right subtrees ‚â§ 1 for all nodes
    • Degenerate Tree: Each parent has only one child (essentially a linked list)

    What is a Binary Search Tree?

    A Binary Search Tree (BST) is a binary tree with a special ordering property: for every node, all values in the left subtree are less than the node's value, and all values in the right subtree are greater. This property enables efficient searching, insertion, and deletion operations in O(log n) time for balanced trees.

    BST Operations

    Operation Average Time Worst Time Description
    Search O(log n) O(n) Compare with root, go left or right recursively
    Insert O(log n) O(n) Find correct position and add as leaf
    Delete O(log n) O(n) Remove node while maintaining BST property
    Min/Max O(log n) O(n) Leftmost node (min) or rightmost node (max)
    Inorder O(n) O(n) Visit all nodes, produces sorted output
    ‚ö†Ô∏è Worst Case: BST degenerates to O(n) when tree becomes unbalanced (e.g., inserting sorted data creates a linked list). Use self-balancing trees (AVL, Red-Black) to guarantee O(log n) operations.

    public class BSTNode
{
    public int Value { get; set; }
    public BSTNode Left { get; set; }
    public BSTNode Right { get; set; }
    
    public BSTNode(int value)
    {
        Value = value;
    }
}

public class BinarySearchTree
{
    private BSTNode root;
    
    // Insert a value
    public void Insert(int value)
    {
        root = InsertRec(root, value);
    }
    
    private BSTNode InsertRec(BSTNode node, int value)
    {
        if (node == null)
            return new BSTNode(value);
        
        if (value < node.Value)
            node.Left = InsertRec(node.Left, value);
        else if (value > node.Value)
            node.Right = InsertRec(node.Right, value);
        
        return node;
    }
    
    // Search for a value
    public bool Search(int value)
    {
        return SearchRec(root, value);
    }
    
    private bool SearchRec(BSTNode node, int value)
    {
        if (node == null)
            return false;
        
        if (value == node.Value)
            return true;
        
        if (value < node.Value)
            return SearchRec(node.Left, value);
        else
            return SearchRec(node.Right, value);
    }
    
    // Delete a value
    public void Delete(int value)
    {
        root = DeleteRec(root, value);
    }
    
    private BSTNode DeleteRec(BSTNode node, int value)
    {
        if (node == null) return null;
        
        if (value < node.Value)
        {
            node.Left = DeleteRec(node.Left, value);
        }
        else if (value > node.Value)
        {
            node.Right = DeleteRec(node.Right, value);
        }
        else
        {
            // Node with one or no child
            if (node.Left == null)
                return node.Right;
            if (node.Right == null)
                return node.Left;
            
            // Node with two children: get inorder successor
            node.Value = MinValue(node.Right);
            node.Right = DeleteRec(node.Right, node.Value);
        }
        
        return node;
    }
    
    // Find minimum value
    private int MinValue(BSTNode node)
    {
        int minValue = node.Value;
        while (node.Left != null)
        {
            minValue = node.Left.Value;
            node = node.Left;
        }
        return minValue;
    }
    
    // Inorder traversal (sorted output)
    public void Inorder()
    {
        InorderRec(root);
    }
    
    private void InorderRec(BSTNode node)
    {
        if (node != null)
        {
            InorderRec(node.Left);
            Console.Write(node.Value + " ");
            InorderRec(node.Right);
        }
    }
    
    // Validate if tree is BST
    public bool IsBST()
    {
        return IsBSTRec(root, int.MinValue, int.MaxValue);
    }
    
    private bool IsBSTRec(BSTNode node, int min, int max)
    {
        if (node == null) return true;
        
        if (node.Value <= min || node.Value >= max)
            return false;
        
        return IsBSTRec(node.Left, min, node.Value) &&
               IsBSTRec(node.Right, node.Value, max);
    }
}

    private BSTNode root;
    
    // Insert a value
    public void insert(int value) {
        root = insertRec(root, value);
    }
    
    private BSTNode insertRec(BSTNode node, int value) {
        if (node == null)
            return new BSTNode(value);
        
        if (value < node.value)
            node.left = insertRec(node.left, value);
        else if (value > node.value)
            node.right = insertRec(node.right, value);
        
        return node;
    }
    
    // Search for a value
    public boolean search(int value) {
        return searchRec(root, value);
    }
    
    private boolean searchRec(BSTNode node, int value) {
        if (node == null)
            return false;
        
        if (value == node.value)
            return true;
        
        if (value < node.value)
            return searchRec(node.left, value);
        else
            return searchRec(node.right, value);
    }
    
    // Delete a value
    public void delete(int value) {
        root = deleteRec(root, value);
    }
    
    private BSTNode deleteRec(BSTNode node, int value) {
        if (node == null) return null;
        
        if (value < node.value) {
            node.left = deleteRec(node.left, value);
        } else if (value > node.value) {
            node.right = deleteRec(node.right, value);
        } else {
            // Node with one or no child
            if (node.left == null)
                return node.right;
            if (node.right == null)
                return node.left;
            
            // Node with two children: get inorder successor
            node.value = minValue(node.right);
            node.right = deleteRec(node.right, node.value);
        }
        
        return node;
    }
    
    // Find minimum value
    private int minValue(BSTNode node) {
        int minValue = node.value;
        while (node.left != null) {
            minValue = node.left.value;
            node = node.left;
        }
        return minValue;
    }
    
    // Inorder traversal (sorted output)
    public void inorder() {
        inorderRec(root);
    }
    
    private void inorderRec(BSTNode node) {
        if (node != null) {
            inorderRec(node.left);
            System.out.print(node.value + " ");
            inorderRec(node.right);
        }
    }
    
    // Find kth smallest element
    public int kthSmallest(int k) {
        int[] count = {0};
        int[] result = {-1};
        kthSmallestHelper(root, k, count, result);
        return result[0];
    }
    
    private void kthSmallestHelper(BSTNode node, int k, int[] count, int[] result) {
        if (node == null) return;
        
        kthSmallestHelper(node.left, k, count, result);
        
        count[0]++;
        if (count[0] == k) {
            result[0] = node.value;
            return;
        }
        
        kthSmallestHelper(node.right, k, count, result);
    }
}

    What is an AVL Tree?

    An AVL Tree (named after inventors Adelson-Velsky and Landis) is a self-balancing binary search tree where the height difference between left and right subtrees (balance factor) of every node is at most 1. This guarantees O(log n) time complexity for all operations, preventing the worst-case O(n) scenario of regular BST.

    Balance Factor

    Balance Factor = Height(Left Subtree) - Height(Right Subtree)

    Valid values: -1, 0, +1. If balance factor becomes < -1 or > +1, the tree is rebalanced using rotations.

    Balanced AVL Tree: All balance factors are -1, 0, or +1

    Rotation Operations

    When balance factor violates AVL property, rotations restore balance:

    Rotation When to Use Description
    Left Rotation (LL) Right-heavy subtree (BF < -1) Rotate left child up to become new root
    Right Rotation (RR) Left-heavy subtree (BF > +1) Rotate right child up to become new root
    Left-Right (LR) Left child is right-heavy Left rotate left child, then right rotate node
    Right-Left (RL) Right child is left-heavy Right rotate right child, then left rotate node

    public class AVLNode
{
    public int Value { get; set; }
    public AVLNode Left { get; set; }
    public AVLNode Right { get; set; }
    public int Height { get; set; }
    
    public AVLNode(int value)

    }
}

public class AVLTree
{
    private AVLNode root;
    
    private int Height(AVLNode node)
    {
        return node == null ? 0 : node.Height;
    }
    
    private int GetBalance(AVLNode node)
    {
        return node == null ? 0 : Height(node.Left) - Height(node.Right);
    }
    
    private void UpdateHeight(AVLNode node)
    {
        node.Height = 1 + Math.Max(Height(node.Left), Height(node.Right));
    }
    
    // Right rotation
    private AVLNode RotateRight(AVLNode y)
    {
        AVLNode x = y.Left;
        AVLNode T2 = x.Right;
        
        x.Right = y;
        y.Left = T2;
        
        UpdateHeight(y);
        UpdateHeight(x);
        
        return x;
    }
    
    // Left rotation
    private AVLNode RotateLeft(AVLNode x)
    {
        AVLNode y = x.Right;
        AVLNode T2 = y.Left;
        
        y.Left = x;
        x.Right = T2;
        
        UpdateHeight(x);
        UpdateHeight(y);
        
        return y;
    }
    
    // Insert and rebalance
    public void Insert(int value)
    {
        root = InsertRec(root, value);
    }
    
    private AVLNode InsertRec(AVLNode node, int value)
    {
        // Standard BST insert
        if (node == null)
            return new AVLNode(value);
        
        if (value < node.Value)
            node.Left = InsertRec(node.Left, value);
        else if (value > node.Value)
            node.Right = InsertRec(node.Right, value);
        else
            return node; // Duplicate values not allowed
        
        // Update height
        UpdateHeight(node);
        
        // Get balance factor
        int balance = GetBalance(node);
        
        // Left-Left case
        if (balance > 1 && value < node.Left.Value)
            return RotateRight(node);
        
        // Right-Right case
        if (balance < -1 && value > node.Right.Value)
            return RotateLeft(node);
        
        // Left-Right case
        if (balance > 1 && value > node.Left.Value)
        {
            node.Left = RotateLeft(node.Left);
            return RotateRight(node);
        }
        
        // Right-Left case
        if (balance < -1 && value < node.Right.Value)
        {
            node.Right = RotateRight(node.Right);
            return RotateLeft(node);
        }
        
        return node;
    }
    
    // Delete and rebalance
    public void Delete(int value)
    {
        root = DeleteRec(root, value);
    }
    
    private AVLNode DeleteRec(AVLNode node, int value)
    {
        if (node == null) return null;
        
        // Standard BST delete
        if (value < node.Value)
            node.Left = DeleteRec(node.Left, value);
        else if (value > node.Value)
            node.Right = DeleteRec(node.Right, value);
        else
        {
            if (node.Left == null || node.Right == null)
            {
                node = (node.Left != null) ? node.Left : node.Right;
            }
            else
            {
                AVLNode temp = MinValueNode(node.Right);
                node.Value = temp.Value;
                node.Right = DeleteRec(node.Right, temp.Value);
            }
        }
        
        if (node == null) return null;
        
        // Update height
        UpdateHeight(node);
        
        // Rebalance
        int balance = GetBalance(node);
        
        if (balance > 1 && GetBalance(node.Left) >= 0)
            return RotateRight(node);
        
        if (balance > 1 && GetBalance(node.Left) < 0)
        {
            node.Left = RotateLeft(node.Left);
            return RotateRight(node);
        }
        
        if (balance < -1 && GetBalance(node.Right) <= 0)
            return RotateLeft(node);
        
        if (balance < -1 && GetBalance(node.Right) > 0)
        {
            node.Right = RotateRight(node.Right);
            return RotateLeft(node);
        }
        
        return node;
    }
    
    private AVLNode MinValueNode(AVLNode node)
    {
        AVLNode current = node;
        while (current.Left != null)
            current = current.Left;
        return current;
    }
}

    What is a Heap?

    A Heap is a specialized complete binary tree that satisfies the heap property: in a max-heap, each parent node is greater than or equal to its children; in a min-heap, each parent is less than or equal to its children. Heaps are commonly implemented using arrays and are the foundation for priority queues and heap sort.

    Array Representation

    Heaps are typically stored in arrays for efficient memory usage:

    • Parent of node at index i: (i-1)/2
    • Left child of node at index i: 2*i + 1
    • Right child of node at index i: 2*i + 2

    Heap Operations

    Operation Time Complexity Description
    Insert (Push) O(log n) Add element at end, heapify up
    Extract Max/Min (Pop) O(log n) Remove root, replace with last, heapify down
    Peek (Top) O(1) Return root without removing
    Build Heap O(n) Convert array to heap (heapify from bottom)
    Heap Sort O(n log n) Build heap, repeatedly extract max/min

    // Min-Heap Implementation
public class MinHeap
{
    private List<int> heap = new List<int>();
    
    private int Parent(int i) => (i - 1) / 2;
    private int LeftChild(int i) => 2 * i + 1;
    private int RightChild(int i) => 2 * i + 2;
    
    private void Swap(int i, int j)
    {
        int temp = heap[i];
        heap[i] = heap[j];
        heap[j] = temp;
    }
    
    // Heapify up (after insertion)
    private void HeapifyUp(int i)
    {
        while (i > 0 && heap[Parent(i)] > heap[i])
        {
            Swap(i, Parent(i));
            i = Parent(i);
        }
    }
    
    // Heapify down (after extraction)
    private void HeapifyDown(int i)
    {
        int minIndex = i;
        int left = LeftChild(i);
        int right = RightChild(i);
        
        if (left < heap.Count && heap[left] < heap[minIndex])
            minIndex = left;
        
        if (right < heap.Count && heap[right] < heap[minIndex])
            minIndex = right;
        
        if (i != minIndex)
        {
            Swap(i, minIndex);
            HeapifyDown(minIndex);
        }
    }
    
    // Insert element
    public void Insert(int value)
    {
        heap.Add(value);
        HeapifyUp(heap.Count - 1);
    }
    
    // Extract minimum
    public int ExtractMin()
    {
        if (heap.Count == 0)
            throw new InvalidOperationException("Heap is empty");
        
        int min = heap[0];
        heap[0] = heap[heap.Count - 1];
        heap.RemoveAt(heap.Count - 1);
        
        if (heap.Count > 0)
            HeapifyDown(0);
        
        return min;
    }
    
    // Peek minimum
    public int Peek()
    {
        if (heap.Count == 0)
            throw new InvalidOperationException("Heap is empty");
        return heap[0];
    }
    
    // Build heap from array
    public void BuildHeap(int[] arr)
    {
        heap = new List<int>(arr);
        
        // Start from last non-leaf node
        for (int i = (heap.Count / 2) - 1; i >= 0; i--)
            HeapifyDown(i);
    }
    
    public int Size => heap.Count;
    public bool IsEmpty => heap.Count == 0;
}

        int temp = heap.get(i);
        heap.set(i, heap.get(j));
        heap.set(j, temp);
    }
    
    // Heapify up
    private void heapifyUp(int i) {
        while (i > 0 && heap.get(parent(i)) > heap.get(i)) {
            swap(i, parent(i));
            i = parent(i);
        }
    }
    
    // Heapify down
    private void heapifyDown(int i) {
        int minIndex = i;
        int left = leftChild(i);
        int right = rightChild(i);
        
        if (left < heap.size() && heap.get(left) < heap.get(minIndex))
            minIndex = left;
        
        if (right < heap.size() && heap.get(right) < heap.get(minIndex))
            minIndex = right;
        
        if (i != minIndex) {
            swap(i, minIndex);
            heapifyDown(minIndex);
        }
    }
    
    // Insert
    public void insert(int value) {
        heap.add(value);
        heapifyUp(heap.size() - 1);
    }
    
    // Extract minimum
    public int extractMin() {
        if (heap.isEmpty())
            throw new IllegalStateException("Heap is empty");
        
        int min = heap.get(0);
        heap.set(0, heap.get(heap.size() - 1));
        heap.remove(heap.size() - 1);
        
        if (!heap.isEmpty())
            heapifyDown(0);
        
        return min;
    }
    
    // Peek
    public int peek() {
        if (heap.isEmpty())
            throw new IllegalStateException("Heap is empty");
        return heap.get(0);
    }
    
    public int size() { return heap.size(); }
    public boolean isEmpty() { return heap.isEmpty(); }
}

class MinHeap:
    def __init__(self):
        self.heap = []
    
    def parent(self, i):
        return (i - 1) // 2
    
    def left_child(self, i):
        return 2 * i + 1
    
    def right_child(self, i):
        return 2 * i + 2
    
    def insert(self, value):
        self.heap.append(value)
        self._heapify_up(len(self.heap) - 1)
    
    def _heapify_up(self, i):
        while i > 0 and self.heap[self.parent(i)] > self.heap[i]:
            # Swap with parent
            parent_idx = self.parent(i)
            self.heap[i], self.heap[parent_idx] = self.heap[parent_idx], self.heap[i]
            i = parent_idx
    
    def extract_min(self):
        if not self.heap:
            raise IndexError("Heap is empty")
        
        min_val = self.heap[0]
        self.heap[0] = self.heap[-1]
        self.heap.pop()
        
        if self.heap:
            self._heapify_down(0)
        
        return min_val
    
    def _heapify_down(self, i):
        min_index = i
        left = self.left_child(i)
        right = self.right_child(i)
        
        if left < len(self.heap) and self.heap[left] < self.heap[min_index]:
            min_index = left
        
        if right < len(self.heap) and self.heap[right] < self.heap[min_index]:
            min_index = right
        
        if i != min_index:
            self.heap[i], self.heap[min_index] = self.heap[min_index], self.heap[i]
            self._heapify_down(min_index)
    
    def peek(self):
        if not self.heap:
            raise IndexError("Heap is empty")
        return self.heap[0]
    
    def size(self):
        return len(self.heap)

    A Graph is a non-linear data structure consisting of vertices (nodes) connected by edges (links). Graphs model relationships between entities and are used to solve problems involving networks, paths, connectivity, and dependencies. Unlike trees, graphs can have cycles and multiple paths between nodes.

            System.out.print(vertex + " ");

                if (!visited.contains(neighbor)) {
                    visited.add(neighbor);
                    queue.offer(neighbor);
                }
            }
        }
        System.out.println();
    }

    public void dfs(int start) {
        Set<Integer> visited = new HashSet<>();
        dfsHelper(start, visited);
        System.out.println();
    }
    
    private void dfsHelper(int vertex, Set<Integer> visited) {
        visited.add(vertex);
        System.out.print(vertex + " ");
        
        for (int neighbor : adjList.get(vertex)) {
            if (!visited.contains(neighbor))
                dfsHelper(neighbor, visited);
        }
    }
    
    // Topological Sort (DAG only)
    public List<Integer> topologicalSort() {
        Map<Integer, Integer> inDegree = new HashMap<>();
        
        // Calculate in-degrees
        for (int vertex : adjList.keySet()) {
            inDegree.putIfAbsent(vertex, 0);
            for (int neighbor : adjList.get(vertex)) {
                inDegree.put(neighbor, inDegree.getOrDefault(neighbor, 0) + 1);
            }
        }
        
        // Queue vertices with 0 in-degree
        Queue<Integer> queue = new LinkedList<>();
        for (int vertex : inDegree.keySet()) {
            if (inDegree.get(vertex) == 0)
                queue.offer(vertex);
        }
        
        List<Integer> result = new ArrayList<>();
        
        while (!queue.isEmpty()) {
            int vertex = queue.poll();
            result.add(vertex);
            
            for (int neighbor : adjList.get(vertex)) {
                inDegree.put(neighbor, inDegree.get(neighbor) - 1);
                if (inDegree.get(neighbor) == 0)
                    queue.offer(neighbor);
            }
        }
        
        return result.size() == adjList.size() ? result : new ArrayList<>();
    }

            int vertex = queue.poll();
            int dist = distances.get(vertex);

                if (!distances.containsKey(neighbor)) {
                    distances.put(neighbor, dist + 1);
                    queue.offer(neighbor);
                }
            }
        }
        
        return distances;
    }

    from collections import deque, defaultdict

class Graph:
    def __init__(self, directed=False):
        self.adj_list = defaultdict(list)
        self.is_directed = directed
    
    # Add vertex
    def add_vertex(self, vertex):
        if vertex not in self.adj_list:
            self.adj_list[vertex] = []
    
    # Add edge
    def add_edge(self, from_vertex, to_vertex):
        self.adj_list[from_vertex].append(to_vertex)
        
        if not self.is_directed:
            self.adj_list[to_vertex].append(from_vertex)
    
    # Get neighbors
    def get_neighbors(self, vertex):
        return self.adj_list.get(vertex, [])
    
    # Breadth-First Search (BFS)
    def bfs(self, start):
        visited = set()
        queue = deque([start])
        visited.add(start)
        result = []
        
        while queue:
            vertex = queue.popleft()
            result.append(vertex)
            
            for neighbor in self.adj_list[vertex]:
                if neighbor not in visited:
                    visited.add(neighbor)
                    queue.append(neighbor)
        
        return result
    
    # Depth-First Search (DFS)
    def dfs(self, start):
        visited = set()
        result = []
        
        def dfs_helper(vertex):
            visited.add(vertex)
            result.append(vertex)
            
            for neighbor in self.adj_list[vertex]:
                if neighbor not in visited:
                    dfs_helper(neighbor)
        
        dfs_helper(start)
        return result
    
    # Check if path exists
    def has_path(self, start, target):
        visited = set()
        
        def dfs_path(vertex):
            if vertex == target:
                return True
            if vertex in visited:
                return False
            
            visited.add(vertex)
            
            for neighbor in self.adj_list[vertex]:
                if dfs_path(neighbor):
                    return True
            
            return False
        
        return dfs_path(start)
    
    # Find all paths
    def find_all_paths(self, start, target):
        paths = []
        
        def dfs_all_paths(vertex, path):
            path.append(vertex)
            
            if vertex == target:
                paths.append(path.copy())
            else:
                for neighbor in self.adj_list[vertex]:
                    if neighbor not in path:  # Avoid cycles
                        dfs_all_paths(neighbor, path)
            
            path.pop()
        
        dfs_all_paths(start, [])
        return paths
    
    # Shortest path (unweighted)
    def shortest_path(self, start, target):
        visited = {start}
        queue = deque([(start, [start])])
        
        while queue:
            vertex, path = queue.popleft()
            
            if vertex == target:
                return path
            
            for neighbor in self.adj_list[vertex]:
                if neighbor not in visited:
                    visited.add(neighbor)
                    queue.append((neighbor, path + [neighbor]))
        
        return None
    
    # Detect cycle (undirected graph)
    def has_cycle(self):
        visited = set()
        
        def dfs_cycle(vertex, parent):
            visited.add(vertex)
            
            for neighbor in self.adj_list[vertex]:
                if neighbor not in visited:
                    if dfs_cycle(neighbor, vertex):
                        return True
                elif neighbor != parent:
                    return True  # Back edge
            
            return False
        
        for vertex in self.adj_list:
            if vertex not in visited:
                if dfs_cycle(vertex, None):
                    return True
        
        return False
    
    # Topological Sort (DAG only)
    def topological_sort(self):
        in_degree = {vertex: 0 for vertex in self.adj_list}
        
        # Calculate in-degrees
        for vertex in self.adj_list:
            for neighbor in self.adj_list[vertex]:
                in_degree[neighbor] = in_degree.get(neighbor, 0) + 1
        
        # Queue vertices with 0 in-degree
        queue = deque([v for v in in_degree if in_degree[v] == 0])
        result = []
        
        while queue:
            vertex = queue.popleft()
            result.append(vertex)
            
            for neighbor in self.adj_list[vertex]:
                in_degree[neighbor] -= 1
                if in_degree[neighbor] == 0:
                    queue.append(neighbor)
        
        return result if len(result) == len(self.adj_list) else []
    
    # Connected components
    def connected_components(self):
        visited = set()
        components = []
        
        def dfs_component(vertex, component):
            visited.add(vertex)
            component.append(vertex)
            
            for neighbor in self.adj_list[vertex]:
                if neighbor not in visited:
                    dfs_component(neighbor, component)
        
        for vertex in self.adj_list:
            if vertex not in visited:
                component = []
                dfs_component(vertex, component)
                components.append(component)
        
        return components
    
    # Print graph
    def print_graph(self):
        for vertex, neighbors in self.adj_list.items():
            print(f"{vertex} -> {neighbors}")

    Graph Traversal Algorithms

    Breadth-First Search (BFS)

    • Explores graph level by level using a queue
    • Finds shortest path in unweighted graphs
    • Time: O(V + E), Space: O(V)
    • Use Cases: Shortest path, level-order traversal, peer-to-peer networks

    Depth-First Search (DFS)

    • Explores as far as possible before backtracking (using recursion/stack)
    • Good for detecting cycles and topological sorting
    • Time: O(V + E), Space: O(V)
    • Use Cases: Cycle detection, topological sort, maze solving, path finding

    A Hash Table (also called Hash Map or Dictionary) is a data structure that maps keys to values using a hash function. It provides average O(1) time complexity for insertions, deletions, and lookups, making it one of the most efficient data structures for key-value storage and retrieval.

    1. Hash Function

    Converts a key into an integer index:

    • Must be deterministic (same key ‚Üí same hash)
    • Should distribute keys uniformly to minimize collisions
    • Should be fast to compute
    • Example: index = hashCode(key) % arraySize

    Each bucket stores a linked list (or other collection) of entries.

    • ‚úÖ Simple to implement
    • ‚úÖ Never fills up (unbounded)
    • ‚ùå Extra memory for pointers
    • ‚ùå Cache unfriendly

    Load Factor = Number of Entries / Table Size

public class SimpleHashTable<TKey, TValue>
{
    private class Entry
    {
        public TKey Key { get; set; }
        public TValue Value { get; set; }
        public Entry Next { get; set; }
        
        public Entry(TKey key, TValue value)
        {
            Key = key;
            Value = value;
        }
    }
    
    private Entry[] buckets;
    private int count;
    private const double LoadFactorThreshold = 0.75;
    
    public SimpleHashTable(int capacity = 16)
    {
        buckets = new Entry[capacity];
        count = 0;
    }
    
    private int GetBucketIndex(TKey key)
    {
        int hashCode = key.GetHashCode();
        return Math.Abs(hashCode) % buckets.Length;
    }
    
    public void Put(TKey key, TValue value)
    {
        if (key == null)
            throw new ArgumentNullException(nameof(key));
        
        // Check if resize needed
        if ((double)count / buckets.Length > LoadFactorThreshold)
            Resize();
        
        int index = GetBucketIndex(key);
        Entry current = buckets[index];
        
        // Update existing key
        while (current != null)
        {
            if (current.Key.Equals(key))
            {
                current.Value = value;
                return;
            }
            current = current.Next;
        }
        
        // Insert new entry at beginning
        Entry newEntry = new Entry(key, value)
        {
            Next = buckets[index]
        };
        buckets[index] = newEntry;
        count++;
    }
    
    public TValue Get(TKey key)
    {
        if (key == null)
            throw new ArgumentNullException(nameof(key));
        
        int index = GetBucketIndex(key);
        Entry current = buckets[index];
        
        while (current != null)
        {
            if (current.Key.Equals(key))
                return current.Value;
            current = current.Next;
        }
        
        throw new KeyNotFoundException($"Key '{key}' not found");
    }
    
    public bool ContainsKey(TKey key)
    {
        if (key == null) return false;
        
        int index = GetBucketIndex(key);
        Entry current = buckets[index];
        
        while (current != null)
        {
            if (current.Key.Equals(key))
                return true;
            current = current.Next;
        }
        
        return false;
    }
    
    public bool Remove(TKey key)
    {
        if (key == null) return false;
        
        int index = GetBucketIndex(key);
        Entry current = buckets[index];
        Entry prev = null;
        
        while (current != null)
        {
            if (current.Key.Equals(key))
            {
                if (prev == null)
                    buckets[index] = current.Next;
                else
                    prev.Next = current.Next;
                
                count--;
                return true;
            }
            prev = current;
            current = current.Next;
        }
        
        return false;
    }
    
    private void Resize()
    {
        Entry[] oldBuckets = buckets;
        buckets = new Entry[oldBuckets.Length * 2];
        count = 0;
        
        foreach (Entry entry in oldBuckets)
        {
            Entry current = entry;
            while (current != null)
            {
                Put(current.Key, current.Value);
                current = current.Next;
            }
        }
    }
    
    public int Count => count;
}

                if (prev == null)

                else
                    prev.next = current.next;
                
                size--;

            }
            prev = current;
            current = current.next;
        }

class SimpleHashTable:
    class Entry:
        def __init__(self, key, value):
            self.key = key
            self.value = value
            self.next = None
    
    def __init__(self, capacity=16):
        self.buckets = [None] * capacity
        self.size = 0
        self.load_factor_threshold = 0.75
    
    def _get_bucket_index(self, key):
        return hash(key) % len(self.buckets)
    
    def put(self, key, value):
        # Resize if needed
        if self.size / len(self.buckets) > self.load_factor_threshold:
            self._resize()
        
        index = self._get_bucket_index(key)
        current = self.buckets[index]
        
        # Update existing key
        while current:
            if current.key == key:
                current.value = value
                return
            current = current.next
        
        # Insert new entry at beginning
        new_entry = self.Entry(key, value)
        new_entry.next = self.buckets[index]
        self.buckets[index] = new_entry
        self.size += 1
    
    def get(self, key):
        index = self._get_bucket_index(key)
        current = self.buckets[index]
        
        while current:
            if current.key == key:
                return current.value
            current = current.next
        
        raise KeyError(f"Key '{key}' not found")
    
    def contains_key(self, key):
        try:
            self.get(key)
            return True
        except KeyError:
            return False
    
    def remove(self, key):
        index = self._get_bucket_index(key)
        current = self.buckets[index]
        prev = None
        
        while current:
            if current.key == key:
                if prev is None:
                    self.buckets[index] = current.next
                else:
                    prev.next = current.next
                
                self.size -= 1
                return True
            prev = current
            current = current.next
        
        return False
    
    def _resize(self):
        old_buckets = self.buckets
        self.buckets = [None] * (len(old_buckets) * 2)
        self.size = 0
        
        for entry in old_buckets:
            current = entry
            while current:
                self.put(current.key, current.value)
                current = current.next
    
    def __len__(self):
        return self.size
    
    def __setitem__(self, key, value):
        self.put(key, value)
    
    def __getitem__(self, key):
        return self.get(key)
    
    def __contains__(self, key):
        return self.contains_key(key)
    
    def __delitem__(self, key):
        if not self.remove(key):
            raise KeyError(f"Key '{key}' not found")

    What is a Set?

    A Set is a collection that stores unique elements with no duplicates. Sets support mathematical operations like union, intersection, and difference, making them ideal for membership testing, deduplication, and set theory operations. Most implementations use hash tables internally for O(1) operations.

    Set Properties

    • Uniqueness: No duplicate elements allowed
    • Unordered: Elements have no specific order (in HashSet)
    • Fast Lookup: O(1) average time to check membership
    • Mutable: Can add/remove elements (in most implementations)

    Types of Sets

    üî∑ HashSet

    Unordered set using hash table

    • O(1) add, remove, contains
    • No ordering guarantees
    • Best for general use

    üî∑ TreeSet / SortedSet

    Ordered set using balanced tree

    • O(log n) add, remove, contains
    • Elements sorted
    • Range queries supported

    üî∑ LinkedHashSet

    Maintains insertion order

    • O(1) add, remove, contains
    • Insertion order preserved
    • Predictable iteration

    ‚ùå Wrong Data Structure Choice

    ‚ùå Not Considering Thread Safety

    // ‚ùå Bad: Sharing mutable collection across threads

// Multiple threads modifying can cause corruption

// ‚úÖ Good: Use thread-safe collections

// Or use locks/synchronization

    ‚ùå Modifying Collection During Iteration

    // ‚ùå Bad: Modifying while iterating

{
    if (item > 10)
        list.Remove(item); // Throws exception!
}

// ‚úÖ Good: Use ToList() or iterate backwards
foreach (var item in list.ToList())
{
    if (item > 10)
        list.Remove(item);
}

// Or remove using predicate
list.RemoveAll(x => x > 10);

    // ‚ùå Bad: Custom object in HashSet without proper equality
